## Project Objectives
` These are the goals at minimum this project must achieve  `
The primary objectives of this project are to:
• Design and develop a secure booking and payment system
• Support structured class blocks and skill progression
• Provide priority booking and pricing incentives
• Implement a digital classroom with tiered access
• Enable communication, notifications, and enquiries
• Deliver a members-only community area
• Demonstrate competence in full-stack development, database design, and
UX/UI principles based on a RESTful architecture style.
